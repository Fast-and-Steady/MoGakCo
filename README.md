### 컨벤션
**Commit**
- 커밋 메세지
- type :  "yyyy-mm-dd 내용" 
- ex) 2023-03-26 백준 문제 풀이

**merge request**

- 프로젝트가 아닌 개인 공부저장 용도로, Merge를 사용, 스쿼시나 리베이스의 경우 프로젝트 때, 사용한다.
